Matching Zero or More of Any character

  • The period (.) is used to match any single character

    • sort of like $ shell metacharacater
    • for example /s..th/ would
      • match smith, swath, snith
      • would not match snatch (more than 2 chars), sth (no chars), petth
    • . Stands for any valid ASCII character

  • Asterisk (*) matches zero or more of any character

    • /A*/ means match anything with 0 or more X; this actually will match anything
    • /AA*/ means match anything that has an A followed by zero or more As
    • In the last case the first A was constant and had to be there unlike A*
