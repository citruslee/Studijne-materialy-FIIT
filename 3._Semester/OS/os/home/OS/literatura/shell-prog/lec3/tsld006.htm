Matching End Line Anchor Point

  • $ matches a character at the anchored at end of line

    • 1,$s/smith$/adams/
    • changes only smith at end of line and no where else

  • This use of $ different than 1,$ use of it

    • expression match character is in between / /
    • the first $ is before the command s
    • this first $ defines the end of the line range
    • line range always specified: start,end
    • when $ specified as end then means the last line in the file
