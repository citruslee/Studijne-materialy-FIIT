Matching Exact Number of Characters

  • Generally match looks for the largest number of character matching the regular expressions

  • \{min,max\} syntax determines a certain number of chars

  • [A-Z]\{1,5\}

    • matches from 1 to 5 upper case letters
    • for cmd 1,$s/[A-Z]\{1,5\}/p/ would replace AAA with p
    • but BCDEFZW would become pZW since ZW were the 6th and 7th char and match only for up to 5

  • \{number\} matches only exactly that many characters rather than a minimum to maximum e.g. /a\{4\} matches exactly any aaaa and not aa or aaa or aaaba

  • \ char before { and } necessary because it says not to treat these as plain characters but special characters.

  • \ known as the escape character
