Save Pattern Examples

  • \(a\) matches the first a on the line wherever found and puts in register 1

  • \(a\)\1 - stores the first a in register 1 then adds the value in register one to the match pattern so this expression would match aa

    • matches any two of the same characters
    • often used with \(.\) which would allow match of any two of the same character -remember . Means any char

  • registers can be used on the subsitute part of the s command. Characters after the second /

    • 1,$s/\(.*\)tab_char_here\(.*\)/\2 \1/
    • switch all characters before the tab with all the characters and the tab
    • characters before tab put in register 1, chars after tab put in reg. 2
    • the subsitute part of s command: /\2 \1/ says to put what is in reg 2 first and what is in reg. 1 second. So in effect swap
    • very useful for swapping fields in files if fields separated with tab
