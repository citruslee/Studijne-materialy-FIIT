Saving Matched Characters

  • Possible to save a character matched and then use latter in regular expression

    • regular expression mechanism provides up to 9 places to save match -- these known as registers
    • Always saves the pattern in the next register in which nothing has yet been saved. At start this is, of course, register 1

  • This adds a great deal of power and flexibility to regular expression matching but is not used that often except by real “regular expression gurus”

  • To access stored value you just list the desired register
