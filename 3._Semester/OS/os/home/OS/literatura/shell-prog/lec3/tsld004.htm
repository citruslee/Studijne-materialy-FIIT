ed - line editor

  • Examples use ed which is a line editor

    • only works on a single line; must set the line by typing the number
    • : is the prompt for ed
    • don’t see the contents of line unless ask to see with l or p commands

  • Can issue ed commands in vi by being in the command mode and entering the character :

    • will be put at the bottom of the screen
    • the ed colon prompt will appear
    • at this point can enter the ed commands
    • should try the examples in the book this way
