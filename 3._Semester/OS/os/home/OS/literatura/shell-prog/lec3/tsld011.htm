Set Match Examples

  • Put all the characters desired to be matched within the brackets

  • Examples

    • [aAeEiIoOuUwWyY] match any vowel or semi-vowel
    • [bcdfghjklmnopqrstvxz] match lower case consonant but not upper case

  • Example subsititute UNIX for Unix or unix in a text file

    • 1,$s/[Uu]nix/UNIX/g
    • match either U or u
    • the g after last slash indicates EVERY occurrence of the pattern on a single line
    • without g would replace only the first occurrence so if appeared more than once only one replaced
