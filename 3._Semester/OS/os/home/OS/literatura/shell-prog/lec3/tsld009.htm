Character Set and Range Expressions Overview

  • Set and range matching allows you to check for any character in a set or within a range of characters. Used with brackets [ ]

  • Range

    • range for alphabetic is based on ASCII sequence
    • range given by listing start character and the end character with hyphen in between
    • [a-d] match any character between a and d inclusive

  • Set

    • list the individuals characters to be matched
    • any in the set will be matched
    • [aeiouy] match any vowel
