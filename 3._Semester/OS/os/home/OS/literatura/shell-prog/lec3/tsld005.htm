Matching Begin Line Anchor Point

  • Line anchors are

    • beginning of line
    • end of line

  • ^ matches a character at the beginning of line

    • using vi/ed could change each smith at beginning of line with adams by typing on the ed prompt (:)
    • 1,$s/^smith/adams/
    • changes only smiths at the beginning

  • ed specifics

    • s is substitute command
    • 1,$ says from 1 to last
    • if didn’t tell the lines, ed would work only on current line
