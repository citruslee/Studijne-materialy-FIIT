Range Match Examples

  • Matching any letter whether upper or lower case and replace these with ( {ed example}

    • 1,$s/[A-Za-z][A-Za-z]*/)/
    • will match any sequence that starts with a letter and is followed by zero or more letters and replaces these with the )
    • will substitute Acedkie4 with )4.
    • Will not substitute 234444433 because no letters

  • Don’t include spaces or you are including space for the match

    • [a-z0-9] either lower case char or number
    • [a-z 0-9] either lower case, space or number
