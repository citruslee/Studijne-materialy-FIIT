What are Regular Expressions?

  • Special patterns used by some Unix programs to support checking and matching of ASCII character patterns

  • The following programs are able to support matching using regular expressions to more or less degree

    • awk (pattern matching and process language)
    • sed (stream editor)
    • ed (line editor; commands like sed)
    • vi (visual line editor built on ed)
    • grep ( file content pattern finding utility set)
    • Perl and TCL have extensive regular expression support (replacement shell languages)
