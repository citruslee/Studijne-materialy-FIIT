Special Set and Match Syntax

  • If you want to match a hyphen this must be the first character in the set

    • [-A-z] this matches a hyphen and every character between captial Z and lower case z in the ASCII set
    • hyphen first or assumption would be you are trying to specify a range

  • If you want to match every character but the ones specified in the set or range use ^ right after the opening [

    • [^A-Z] match any character except capital letters
    • if ^ not after the opening [, assumption is that you want to match the ^ character
    • [A-Z^a-z] match upper and lower alpha plus ^ char

  • If want to check for set or range at start of line ^ is outside [] - example /^[A-Za-z] match alpha at begin of line
