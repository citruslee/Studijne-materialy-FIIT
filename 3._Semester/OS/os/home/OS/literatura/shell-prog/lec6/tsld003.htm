grep family filters

  • Used to search through file(s) for string values

  • Most useful tool when looking to see whether a file contains a particular sequence of characters.

    • Often used by programmers to look for particular variables in files
    • used by shell programmers to look for which shell a script uses (e.g., based on inclusion of /bin/sh)

  • Key uses

    • invaluable for finding occurences of values in programs
    • finding words in documents
    • selecting parts of the output of a program
