grep Family Members

  • Three members in the family

    • grep search a file(s) for pattern using limited regular expressions
    • egrep search a file(s) for pattern using full regular expressions including use of registers
    • fgrep search a file(s) for a string

  • fgrep the fastest because it does not support use of regular expressions in search

  • Each prints the line when searched value was found and the file in which found
