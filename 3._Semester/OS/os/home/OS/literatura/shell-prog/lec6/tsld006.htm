grep examples

  • who | grep “Apr 11”

    • get who is on system
    • pipe to grep and see who has been on since Apr 11
    • dates put out by who always have two places so if looking for Apr 1 would have to do Apr 1 (extra space in front of 1)

  • ls *| grep ‘[Jj]a’

    • list all files through grep
    • print out only those that have a Ja or ja in their names

  • generally should put the search pattern in single quotes so shell doesn’t interpret characters like * , $, &

    • need to pass these to grep
    • if not in single quotes shell would use them not pass
