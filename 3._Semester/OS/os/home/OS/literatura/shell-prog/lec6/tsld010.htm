Some Additional Examples

  • Delete the 5th line of the file names

      sed 5d names

    • reads first 4 puts on standard output
    • skips the 5th name and then resumes putting the rest
    • input file is NOT altered; only output file has the 5th name missing

  • Print out only the first 10 names and then quit looking at the file names

      sed 10q names

  • Do what tr can do

      sed ‘s/t/T/g’ names

    • same as tr ‘t’ ‘T’ <
    • however tr faster because that only thing it does
