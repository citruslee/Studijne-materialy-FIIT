Using sed Like Other Unix filters

  • Use like grep

    • sed -n ‘/Total/p’ *
    • don’t automatically print lines in file
    • look for Total in each file
    • print the line (p) only if Total appears on it

  • Use like head

    • sed -n ‘1,10p’
    • don’t automatically print lines
    • print only lines 1 through 10
