sed Essentials

  • Derived directly from ed which is the UNIX line editor (large part of which is also included in vi)

  • Used for editing data in script files

  • Don’t access the editor interactively however but in stream mode

  • sed stands for the stream editor

  • commands are applied to EACH line of the specified file

  • pattern determines whether command executed on the line

  • By default the output is placed on the standard output

  • sed automatically outputs each line as it encounters each line in the file

    • unless output suppressed with the -n option
    • then could use the p command to print only when desired
