chmod review

  • chmod is the Unix command that allows setting of permissions

  • it may always be used with octal numbers to indicate the permissions to be set for a file

  • general syntax: chmod permission_indicator file(s)

    • chmod 744 setpass
    • chmod 755 *

  • also allows use of mnemonics rather than octal digits

    • u - user, g-group and o-others
    • r-read, w-write, x-execute
    • use + to add a permission to category or - to subtract
    • chmod u+x setpass adds execute permission for the owner of the file
    • chmod o-w * subtracts w from the world on all files in current directory
