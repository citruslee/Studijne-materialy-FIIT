File Permissions and ls command

  • ls -l gives the permission list

      -rwxr-xr-x 1 stocker nasa 14636 Jan 29 07:56 setpass

      -rw-r--r-- 1 stocker nasa 516 Jan 29 07:46 setpassword.c
      -rw----r-x 1 stocker nasa 406 Nov 9 13:40 temp
      -rw-r--r-- 1 stocker nasa 207 Jan 29 07:46 transform.c

  • Using the setpass as an example

    • user are the leftmost 3 characters: has read, write and execute permissions
    • group are the center 3 characters: has read and execute
    • world are the rightmost 3 characters: has read and execute
    • the leftmost character which is a - is reserved to indicate that file is a directory when a ëdí appears or a link when a ëlí appears.
