Permission Flags

  • Each group is determined by 3 bits (represented by a single octal digit)

  • Each permission type is a single binary bit (on has permission; off donít)

    • 111111111 (1 is on and 0 is off)
    • u g o

  • Within a single group of three digits

    • first bit is read
    • second bit is write
    • third bit is execute

  • So if use an octal number like 754 with chmod

    • 111101100 is the binary representation
    • user has read, write execute
    • group has read and execute
    • others have read
