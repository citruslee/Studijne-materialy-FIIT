Pasting Together File Pieces

  • The inverse of cutting characters or fields from a file is pasting together lines from two or more files

  • Paste does not work on fields or characters but whole lines

  • paste will match line 1 from file1 with line1 from file2 and line2 from file1 with line2 from file 2

  • Again the input file is not changed. The pasted data is placed on standard output

  • paste inserts a tab character between the information taken from one file and the next

    • line1_file1tabline1_file2tabline1-file3 etc
    • more than two files may be pasted together

  • with -d option can specify a separator other than tab
