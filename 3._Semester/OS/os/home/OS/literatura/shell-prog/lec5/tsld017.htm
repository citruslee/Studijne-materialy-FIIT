Translating Characters

  • tr is a filter program that converts one character to another

  • tr only takes its input from standard in

    • must redirect using <
    • or cat and pipe | into tr program

  • general syntax

    • tr from_character to_characters
    • if multiple characters lists in from and to then they are matched from left
    • tr “abc” “def” <
      • a becomes d
      • b becomes e
      • c becomes f
      • does not match string abc with def - works on characters only
