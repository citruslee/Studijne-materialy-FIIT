Cutting fields syntax

  • The general syntax for cutting fields is

      cut -d: -f3,4,7-9 myfile

  • here delimiter set to :
  • fields to be extracted are 3, 4 and 7 through 9

  • If don’t specify the -d option then cut assumes that the tab character will separate fields

    • would type character for a field then hit tab key
    • then the characters for the next field
    • on screen may look like several spaces but really only a single character

  • Just as for characters can list fields separated by comma or can specify a range of fields using the hyphen except single number is just that field (e.g -f1 is not field 1)
