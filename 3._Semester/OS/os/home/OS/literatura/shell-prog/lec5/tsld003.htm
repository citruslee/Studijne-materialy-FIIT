Pipes and Input/Output

  • One of the most powerful tools introduced by Unix is the concept of input/output as just a stream of bytes with no required structure

  • This concept allows the introduction of redirection

    • standard input from file rather than keyboard (<)
    • standard output to file rather than screen (>)

  • Also allows introduction of pipes between programs

    • standard output of program in front of pipe
    • becomes standard input of program following pipe operator (|)

  • Supports flexibility and tying together two programs through data stream
