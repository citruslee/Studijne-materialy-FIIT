Some Available Options for tr

  • Can specify expression ranges to tr

    • tr ‘[A-Z]’ ‘[a-z]’ <
    • translates capital to lower case characters

  • Can use the -s option on tr which takes out multiple occurrences of a character in the to-char part

    • in this use if multiple to char occurs, it leaves one
    • this true whether the to-char already in file or placed in file as a result of the translation
    • very often used to compress out spaces. Change a space to a space but use the -s option and it compresses
      • tr -s ‘ ‘ ‘ ‘<
      • from and to characters the same but result is only single space remains
