Cutting fields

  • Sometimes data is not in fixed format in the file

    • for example name not always 1-15 but varies
    • however,characters are divided into fields that are separated by a delimiter

  • Sample field files with delimiters

    • comma as delimiter

        Stocker Erich,Programmer,236778

        Jones Jim,English Professor,222222
        Washington George,1st President and General,111111

      • fields are name, profession,tax code
      • notice each field can be as big as necessary
      • next field starts after the delimiter
    • passwd is a file with : as a delimiter between fields

  • fields support flexibility because no fixed size
