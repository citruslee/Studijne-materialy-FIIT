Character syntax

  • When specifying the -c option, you can list a range of characters by using the hyphen

    • -c1-9
    • characters 1 through 9

  • Can specify a single number which indicates from that character to the end of line

    • -c15
    • character 15 to end

  • Can also specify a open hyphen to indicate to end of line

    • -c15-

  • Can specify more than one set of character with a list

    • -c1-9,15-19,70
    • chars 1 through 9, and 15 through 19 and 70 through end
