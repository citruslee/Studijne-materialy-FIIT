Summary of Filters Characteristics

  • Share a common design

  • Each writes on stdout the result of its transformation

  • Each can process its data from stdin if there are no arguments given to the command to take data from elsewhere

    • arguments may specify input but not output
    • output may always be sent to pipeline

  • Error messages are written to stderr

    • by default stderr and stdout are the same (screen)
    • stderr output not sent through pipeline which deals with stdout
    • stdout and stderr may be redirected separately
