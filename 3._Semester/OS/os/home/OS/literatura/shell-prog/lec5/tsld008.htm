cut - taking things out of a file

  • Cut extracts things from a file

    • it does not change the file that it is working on
    • extracted items only are placed on standard output

  • cut can extract a certain number of characters if the input data has a fixed format

  • cut can extract a certain number of fields if a common delimiter is used to separate the fields in a file

    • by default cut assumes the tab character is used to separate files
    • user can specify a different delimiter as an argument to cut

  • cut can take its input from standard in via a pipeline
