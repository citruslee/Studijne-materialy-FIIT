Cutting a Fixed Number of Chars

  • If an input file has a fixed format; for example name is always in columns 1-15 of the file, then can use the fixed version of cut

  • fixed format cut uses the -c argument to tell which chars to cut

  • cut -c1-15 myfile

    • tells cut to extract characters 1 through 15 from myfile
    • display the extracted characters on standard output

  • cut -c25 myfile

    • tells cut to extract character 25 from myfile
    • display extracted character on standard output

  • cut -c25- myfile

    • tells cut to extract 25th to end character
