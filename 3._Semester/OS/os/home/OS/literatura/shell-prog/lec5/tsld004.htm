Filters

  • Family of UNIX programs that

    • read some input (often including stdin)
    • perform a simple transformation on it
    • write some output (usually stdout)

  • Examples:

    • grep
    • more
    • sort
    • wc

  • Many of these are the ones which make use of regular expressions

  • Transformations are often an intermediate step to the final desired output

  • Are often included in a pipe line of commands
